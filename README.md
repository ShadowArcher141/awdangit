# awdangit

A basic Discord bot built with Python and `discord.py`.

## Setup

1. Copy `.env.example` to `.env`
2. Set `DISCORD_TOKEN`
3. Install dependencies:

```bash
python -m pip install -r requirements.txt
```

4. Run the bot:

```bash
python bot.py
```

## Commands

- `!ping` - replies with Pong!
